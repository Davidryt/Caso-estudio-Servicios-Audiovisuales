Para ejecutar la herramienta
1.- Colocate en la carpeta que están los archivos

cd /home/labsma/Desktop/Caso-estudio-Servicios-Audiovisuales/Servicio-IPTV-STREAMING/

2.- En este directorio hay una carpeta por cada dispositivo a configurar. Compila el sh para que se configure solo del interior de las carpetas. En el caso de los routers no hace falta acceder a las carpetas, unicamente compilar el sh correspondiente a cada router de el directorio raiz.

3.- Para ejecutar la aplicación, ejecuta una instancia de VideoServer con sh server.sh y otra de sh VideoClient con sh client.sh (Recomendamos iniciar primero el cliente antes que el servidor)
